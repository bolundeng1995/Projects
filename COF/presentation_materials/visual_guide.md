# Visual Guide: COF Trading Strategy Presentation

## Design System

### Color Palette
- **Primary Colors**
  - Deep Blue: #1A237E (Main theme, headers)
  - Teal: #00897B (Accents, highlights)
  - Orange: #FF5722 (Call-to-action, important data)
  - Light Gray: #F5F5F5 (Background)
  - Dark Gray: #212121 (Text)

- **Gradient Combinations**
  - Header Gradient: Deep Blue to Teal
  - Accent Gradient: Teal to Orange
  - Background Gradient: Light Gray to White

### Typography
- **Fonts**
  - Headings: Montserrat Bold
  - Body: Open Sans Regular
  - Code: Fira Code
  - Numbers: Roboto Mono

- **Font Sizes**
  - Title: 44pt
  - Headers: 32pt
  - Subheaders: 24pt
  - Body: 18pt
  - Captions: 14pt

### Icons and Graphics
- **Icon Style**: Material Design Icons
- **Chart Style**: Clean, minimalist with grid lines
- **Diagram Style**: Flow-based with rounded corners

## Presentation Tips

### Visual Hierarchy
1. Use size to indicate importance
2. Apply color strategically
3. Maintain consistent spacing
4. Use white space effectively
5. Guide the eye with design

### Data Visualization
1. Keep it simple
2. Use appropriate chart types
3. Include clear labels
4. Add context
5. Make it interactive

### Animation Usage
1. Use sparingly
2. Keep it smooth
3. Time it well
4. Add purpose
5. Test thoroughly

### Interactive Elements
1. Make them obvious
2. Keep them responsive
3. Add feedback
4. Test usability
5. Have fallbacks
